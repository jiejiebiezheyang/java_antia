#### 介绍
Java学习记录

#### Java概述
- Java是sun公司1995nian推出的，2019年被oracle收购
- Java的爸爸：詹姆斯·高斯林
- Java是一门高级编程语言：语言风格接近人类的自然语言，写程序简单易懂
- Java的流行度很高，商业占用率很高
##### 很重要的特性：可移植性
#### Java能干啥？
什么都可以，但是最被市场认可的是企业级开发
#### Java的技术体系
- JavaSE 标准版：Java技术的核心和基础
- JavaEE 企业版：大型互联网企业级解决方案   充分被市场认可
- JavaME 小型版：移动应用开发的解决方案
